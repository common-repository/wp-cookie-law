Cookie Law Info - New Version , Support latest WP 4.4.2. Allow you to show how your website complies with the EU Cookie Law. Implied Consent. Style it to match your own website.



1. Download plugin zip file to your computer and extract in a folder
1. Upload `wp-cookie-law` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
